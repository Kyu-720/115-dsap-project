# 文字搜尋系統

## Proposal Report

### 動機與目標
動機：實作一個文字搜尋系統，即使只記得部分字、或用複雜條件查詢，也能快速得到相關結果。
目標：布林搜尋與前綴搜尋，結合關鍵字排序。

### 預期功能
1. 關鍵字搜尋
2. 搜尋方法：
- Boolean Search(布林搜尋)
- Prefix tree(前綴詞搜尋)
3. 關鍵字相關性與排序

### 使用技術
Python

### 時程規劃
week 7 : 關鍵字搜尋  
week 8 : Boolean Search  
week 9 : Boolean Search  
week 10 : Prefix tree  
week 11 : Prefix tree  
week 12 : 關鍵字相關性  
week 13 : 排序  
week 14 : 統整合併  

### 與課程的關聯
1. 資料結構 
- Trie tree(字典樹) : 快速搜尋拼字不完整的單字。
- Hash(雜湊表) : 做inverted index（倒排索引），加快搜尋時間
2. 演算法
- Boolean Search(布林搜尋) : 用集合運算：& / |，實作 AND / OR
- Ranking(排序) : 對結果進行排序，讓較相關的文件排序在前
